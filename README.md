# tennis-invaders-releitura-de-space-invaders-
Fiz este jogo para uma aula da graduação em Análise de Desenvolvimento de Sistemas, usando a linguagem Scratch. Foi bom para entender um pouco mais sobre lógica de programação desde a base, e também para entrar no mundo da programação de uma forma um pouco mais recreativa. :D

O jogo funciona da seguinte forma:
-> Você pilota uma nave espacial de pixel art.
-> Você tem 3 "vidas".
-> Caso o inimigo ( vermelho ) encoste na sua nave, você perde uma vida. Se perder todas, o jogo acaba.

-> Caso você consiga ( sem perder a nave 3x ) atingir os inimigos
( tanto o da direita quanto o da esquerda ) 10x seguidas, você ganha uma bola de tênis,
 que aparece logo abaixo dos corações, que estão em cima da tela.

 -> Ganhando 3 bolas de tênis, você VENCE O JOGO.

 É um jogo simples, portanto funcional e inteligente. É o meu começo no mundo da programação, e também na gradução ( apenas a primeira aula de desenvolvimento de jogos ),
 mas que já foi o suficiente para treinar o bastante lógica de programação.

 É isso! :D
